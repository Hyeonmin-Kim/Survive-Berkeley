## How to set up
1. Open terminal ("terminal 1") and type `cd client`
2. Type `npm install`
3. Open new terminal ("terminal 2") and type `cd server`
4. Type `npm install`
5. Type `node app.js`
6. Return to terminal 1 and type `npm run dev`
7. Get into the localhost url
